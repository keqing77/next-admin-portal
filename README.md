## QuickStart

```shell
# copy env file from template
cp .env.example .env.local

# create .npmrc file and add nexus registry
echo "registry=<URL>" > .npmrc

# install dependencies
pnpm i

# start the local server
pnpm run dev
```

## Using Docker

```
docker pull
docker run
```
